코딩
- 파일 추가 시 camel case로 제목 작성변수 생성 및 할당 시 코드 공유 (메모장같은 툴에 목록으로 작성)
- 변수 역시도 일단 camel case로
- 작성코드 작성 시 기능적인 부분에 주석으로 어떤 기능인지 설명을 달아주기
- 오류 코드가 발생하면 오류 공유하기
- 커밋 타이틀은 부모 컴포넌트 제목으로 (커밋 내용도 부모-자식 계층순으로 내용 정리하여 올리기)
